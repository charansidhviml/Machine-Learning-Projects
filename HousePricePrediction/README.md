House Price Prediction using Machine Learning

Project Overview

This project predicts house prices using the Ames Housing dataset from the Kaggle House Prices competition. The complete machine learning workflow includes data preprocessing, feature engineering, model building, hyperparameter tuning, and performance evaluation.

The objective is to compare multiple regression algorithms and identify the best-performing model for predicting house prices.

 Dataset

* Source: Kaggle House Prices Competition
* Training Data: 1,460 houses
* Target Variable: SalePrice

Project Workflow

* Data Loading
* Exploratory Data Analysis (EDA)
* Missing Value Handling
* Feature Engineering
* One-Hot Encoding
* Feature Alignment
* Train-Validation Split
* Model Training
* Model Evaluation
* Hyperparameter Tuning (GridSearchCV)
* Final Kaggle Submission

 Machine Learning Algorithms Used

* Linear Regression
* Decision Tree Regressor
* K-Nearest Neighbors (KNN) Regressor
* Random Forest Regressor

Evaluation Metrics

* Mean Absolute Error (MAE)
* R² Score


After comparing multiple regression algorithms, Random Forest Regressor achieved the best performance on the validation dataset.
